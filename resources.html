Step-by-Step Checklists

These checklists distill each phase of the incident response lifecycle into actionable steps. Available in PDF and Markdown formats.

  • Prepare Building IR capabilities, team structure, communication plans, and tooling before an incident occurs.
    PDF Markdown
  • Detect Establishing detection sources, hunting for threats, and refining detection capabilities.
    PDF Markdown
  • Verify and Triage Validating a potential incident, assessing risk, and determining response priorities.
    PDF Markdown
  • Response Actions Loop Managing the iterative cycle of scoping, containment, eradication, and recovery.
    PDF Markdown
  • Scope Determining the full extent of compromise across the environment.
    PDF Markdown
  • Contain Stopping attacker activity, preserving evidence, and preventing further harm.
    PDF Markdown
  • Eradicate Investigating attacker presence, removing persistence mechanisms, and remediating root causes.
    PDF Markdown
  • Recovery Validating, testing, and coordinating the return of systems to production.
    PDF Markdown
  • Debrief Documenting the incident, capturing lessons learned, and driving organizational improvement.
    PDF Markdown
